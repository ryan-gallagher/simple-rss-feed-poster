# Simple RSS Feed Poster

A WordPress plugin that automatically creates daily digest posts from an RSS feed.

## Features

- Scheduled daily posting at a configurable time
- Alphabetical sorting of links
- Duplicate detection to prevent reposting
- Title cleanup with customizable prefix replacements
- Auto-strip suspicious prefixes (hostnames, URL fragments)
- Live preview in admin

## Installation

1. Download the plugin file
